# Ext Contrast

Bu proje, HTML/CSS/JavaScript ile yazilmis bir Chrome extension'dir.

Ozellikler:

- Sayfadaki kontrast temasini degistirir.
- Yazilari yuzde 100 ile 200 arasinda buyutur.
- Arial, Verdana, Tahoma gibi daha okunakli yazi tiplerine gecebilir.
- Secilen ayarlari tarayicida saklar.

Kurulum:

1. Chrome'da `chrome://extensions` sayfasini ac.
2. Sag ustten `Developer mode` secenegini aktif et.
3. `Load unpacked` butonuna tikla.
4. Bu klasoru sec: `/Users/anilyilmaz/Desktop/nest/ext-contrast`

Kullanim:

1. Uzanti ikonuna tikla.
2. `Ozellikleri etkinlestir` secenegi ile ac/kapat.
3. `Kontrast modu` listesinden tema sec.
4. `Yazi boyutu` kaydiricisi ile metni buyut.
